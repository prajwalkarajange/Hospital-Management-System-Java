# 🏥 Hospital Management System

A console-based **Hospital Management System** developed using **Java**, **JDBC**, and **MySQL**. The project allows users to manage patients, doctors, and appointments through a menu-driven interface while storing all records in a MySQL database.

---

## 📌 Features

- ✅ Add New Patients
- ✅ View Patient Records
- ✅ View Doctor Records
- ✅ Book Appointments
- ✅ Check Doctor Availability
- ✅ MySQL Database Integration using JDBC
- ✅ Console-based User Interface

---

## 🛠️ Technologies Used

- Java
- JDBC
- MySQL
- IntelliJ IDEA
- Object-Oriented Programming (OOP)

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/Hospital-Management-System-Java.git
```

2. Open the project in IntelliJ IDEA.

3. Add the MySQL JDBC Driver.

4. Create a MySQL database named **hospital**.

5. Import the SQL tables.

6. Update your database username and password.

7. Run

```
HospitalManagementSystem.java
```
